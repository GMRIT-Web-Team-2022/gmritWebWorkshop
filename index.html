Images in HTML

Images can be displayed in browser using img tag. It has attributes like src, alt, height, width. alt is means alternative, We give text if image resource not found or image not rendered properly due to internet issues this text will be displayed, And this also help to improve accessibility. src is to give the source of image.

Example

Note

  • Common mistake we make is giving local file path like 'C:/some/project/images/montains.jpg'. In local it works. but once deployed it breaks we should always give file directory respective to the project folder

Forms in HTML5

Forms is used to collect data from users. And form can have various types of inputs like text, email, phone number, file...etc. Form had attributes get and post which are used to send data. Using get we send data by appending data into url. Using post we send data in HTTP and data is not shown in URL unlike get. action is attribute in which we pass the file or resource which handle form backend.

classes and id's are used in styling elements with name. Classname can be used in multiple elements. Whereas id is unique for element. for classname we use . , for id we use # in css file

  • External : Styles are created in new file with .css extensions and linked to html page

Selectors in CSS

We can select an element using:

Universal selector: using star we select all elements. Class selector: Selecting an element using it's class.
Id Selector: Selecting an element using it's id name.
By tag: Selecting an element using it's tag name.
By Group: Selecting elements using group.

Note

  • Don't use universal selector often. Because it effects each and every element. And Many don't recommend using it.
  • Don't reuse id.
